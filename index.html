Guess the Number Game

Project Description

The player tries to guess a random number within a certain range. The system gives feedback — “too high”, “too low”, or “correct” — until the player finds the right number.

Features

Basic Features

  • Generate a random number (1–100)
  • Input box and “Submit” button
  • Feedback message: too high / too low / correct

Enhanced Features

  • Score tracking (count number of attempts)
  • Limited attempts (lose if you exceed max tries)
  • Hint system (e.g., “very close”)

Advanced Features (Optional)

  • Leaderboard (using local storage)
  • Difficulty levels (change range and attempts)
